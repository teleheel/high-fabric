Tweaks, utility and mechanics

This page contains an overview of the different tweaks, utilities and mechanics that are added by the mods that includes a lot of different, often unrelated stuff.

Gameplay

Type Description Mod
Mechanic Tweak Drop More Experience On Death: Players drop 80% of their experience when dying instead of basically nothing. Fabrication
Mechanic Tweak CanHit: Adds a new CanHit tag that affects melee attacks, bows, crossbows, and tridents. Also works on arrows and fireworks, in which case the restrictions will be AND'd with the restrictions of the bow or crossbow doing the firing. Click the link to learn more. Fabrication
Mechanic Tweak Crawling: Adds a key to explicitly enter the "crawling" stance. No more smushing yourself with a trapdoor. Needed on both the server and client, but doesn't break vanilla clients. Honors the vanilla "Sneak" Accessibility Option. Interacts with Utility > Toggle Stance. Fabrication
Item Tweak Tridents Activate Levers: When a trident hits a lever it will toggle it. Fabrication
Item Tweak Unsaddle Creatures: Shift right click with an empty hand to retrieve a saddle from a pig/strider. Fabrication
Item Tweak Bottled Air: Filling a glass bottle underwater refills 1 air bubble. Fabrication
Item Tweak Tools in Bundles: Allows storing up to 8 nonstackable items in a bundle, as long as nothing else is stored. Fabrication
Item Tweak Blaze Powder Grows Nether Wart: Blaze powder behaves as bone meal for nether wart. Fabrication
Item Tweak Faulty Shields: Shields half non-projectile damage instead of blocking it. Fabrication
Item Tweak betterBonemeal: Bonemeal can be used to grow sugarcane, cactus and lily pads. Carpet Extra
Block Tweak Extract Furnace Experience: Allows clicking on the empty output slot of a furnace to extract the experience, instead of needing to break it. Useful if extracting output items with e.g. a hopper. Fabrication
Block Tweak Bush Walk Doesn't Hurt When Sneaking: Makes walking through berry bushes when sneaking not deal damage. Fabrication
Block Tweak Campfires Ignite Entities: Campfires will set fire to mobs standing on them, therefore also cooking them. Fabrication
Block Tweak Campfires Place Unlit: Campfires are unlit when placed and must be lit with a Flint and Steel. Fabrication
Block Tweak Flammable Cobwebs: Cobwebs can burn. Fabrication
Block Tweak Less Restrictive Note Blocks: Allows note blocks to play if any block next to them has a nonsolid face, instead of only if the block above is air. On the client, just adjusts the note particle to fly the right direction. Fabrication
Block Tweak Nether Cauldron: Water evaporates when being placed inside a cauldron in the nether. Fabrication
Block Tweak No Sneak Bypass: Blocks which would normally not affect players when sneaking (magma blocks) will. Berry bushes will still not affect players when sneaking (as intended). Fabrication
Block Tweak Play Note Blocks In Creative: Allows sneaking when punching note blocks to play them in Creative mode. On the client, prevents the break effect from occurring as usually happens with server-side break prevention. Fabrication
Block Tweak Reverse Note Block Tuning: Sneaking while tuning a note block reduces its pitch rather than increases. Fabrication
Block Tweak Cactus Punching Hurts: Breaking cactus with your hand will damage you the same as hugging it. Fabrication
Block Tweak Exact Note Block Tuning: Right-clicking a note block with a stack of sticks sets its pitch to the size of the stack minus one. Fabrication
Block Tweak Gradual Block Breaking: Snow layers and slabs will be mined a layer at a time. Fabrication
Block Tweak Mechanism Muffling: Placing a block of wool adjacent to a dispenser, dropper, or piston makes it silent. Fabrication
Block Tweak Spreadable Moss: Using bone meal on a stone block with a moss block nearby grows moss to that block, to make it a bit easier to work with for building. Fabrication
Block Tweak Directional Detector Rails: Placing magenta glazed terracotta under a detector rail makes it only detect minecarts moving in the direction of the arrow. Fabrication
Block Tweak Directional Powered Rails: Placing magenta glazed terracotta under a powered rail makes it push minecarts in the direction of the arrow, even from a stop. They also work as hard stops for carts going the wrong direction. Fabrication
Block Tweak chainStone: Chains will stick to each other on the long ends and will stick to other blocks that connect to them directly. Carpet
Block Tweak desertShrubs: Saplings turn into dead shrubs in hot climates and no water access. Carpet
Block Tweak missingTools: Glass can be broken faster with pickaxes. Carpet
Block Tweak movableAmethyst: Allows Budding Amethyst blocks to be moved. Allow for them to be moved by pistons as well as adds extra drop when mining with silk touch pickaxe. Carpet
Block Tweak renewableBlackstone: Nether basalt generator without soul sand below will convert into blackstone instead. Carpet
Block Tweak renewableCoral: Coral structures will grow with bonemeal from coral plants. Carpet
Block Tweak renewableDeepslate: Lava and water generate deepslate and cobbled deepslate instead below Y0. Carpet
Block Tweak emptyShulkerBoxStackAlways: Empty Shulker Boxes will always stack, even inside inventories. Carpet Extra
Anvil Tweak Anvil Repair: Allows right-clicking on an anvil with a Block of Iron to repair it one stage. Fabrication
Anvil Tweak Anvil No Level Limit: Anvils don't become "Too Expensive". Fabrication
Anvil Tweak Anvil Rename Always Costs One: Makes renaming an item on an anvil always cost one level. Fabrication
Anvil Tweak renewableIce: 3 ice blocks stacked ontop of eachother make denser ice when crushed by falling anvils. Frosted turns into normal, normal into packed and packed into blue. Carpet Extra
Anvil Tweak renewableSand: Cobblestone crushed by falling anvils makes sand. Carpet Extra
Dispenser Tweak dispensersCarvePumpkins: Dispensers containing shears can carve pumpkins. Carpet Extra
Dispenser Tweak dispensersFeedAnimals: Dispensers can feed animals if given their breeding item. Can also feed flowers to brown mooshrooms to give them a suspicious stew effect. Carpet Extra
Dispenser Tweak dispensersFillMinecarts: Minecarts can be filled with hoppers, chests, tnt and furnace. Carpet Extra
Dispenser Tweak dispensersMilkAnimals: Dispensers with empty buckets can milk cows/mooshrooms/goats, and get mushroom/suspicious stew from mooshrooms with bowls. Carpet Extra
Dispenser Tweak dispensersPlayRecords: Dispensers can play records if there's a jukebox in front of them. If a record already exists in the jukebox, it gets placed back in the dispenser. Carpet Extra
Dispenser Tweak dispensersPotPlants: Dispensers can put flowers in flower pots. Carpet Extra
Dispenser Tweak dispensersStripBlocks: Dispensers with axes can strip blocks. Can strip logs, remove oxidation, and remove wax. Carpet Extra
Dispenser Tweak dispensersTillSoil: Dispensers with hoes can till soil. Carpet Extra
Dispenser Tweak dispensersUseCauldrons: Dispensers can empty/fill cauldrons with buckets or bottles, and undye leather armor/shulker boxes/banners. Carpet Extra
Dispenser Tweak renewableEndstone: Dragon's breath from dispensers convert cobblestone to end stone. Carpet Extra
Dispenser Tweak renewableNetherrack: Fire charges from dispensers convert cobblestone to netherrack. Carpet Extra
Enchantment Tweak Feather Falling No Trample: Farmland will not get trampled when wearing feather falling. Fabrication
Enchantment Tweak Tridents In Void Return: Makes Loyalty tridents immune to void damage, and causes them to start their return timer upon falling into the void. Fabrication
Enchantment Tweak Channeling Two: Makes Channeling II a valid enchant that works while raining as well. Fabrication
Enchantment Tweak Feather Falling V: Makes Feather Falling V a valid enchant that completely negates fall damage. Fabrication
Enchantment Tweak Feather Falling V Damages Boots: Makes absorbing fall damage with Feather Falling V cause damage to the boots. Fabrication
Enchantment Tweak Fire Aspect Is Flint And Steel: Right-clicking a block with no action with a Fire Aspect tool emulates a click with flint and steel, allowing you to light fires and such with a Fire Aspect tool instead of having to carry around flint and steel. Since it emulates clicking with a Flint and Steel, it's highly compatible. Fabrication
Enchantment Tweak InfiBows: Makes Infinity bows not require an arrow in your inventory to fire. If not present on client, firing bows with no arrows is a bit janky. Fabrication
Enchantment Tweak Bedrock-Like Impaling: Makes the Impaling enchantment act like it does in Bedrock Edition and Combat Test 4. Namely, it deals bonus damage to anything that is in water or rain (i.e. is wet), instead of only aquatic mobs. Interacts with Enhanced Moistness. Fabrication
Enchantment Tweak Infinity Crossbows : Allow putting Infinity on crossbows. Only works for plain arrows. Honors InfiBows. Fabrication
Enchantment Tweak Tridents Accept Power: Allows tridents to accept the Power enchantment, increasing their ranged damage. It's pitiful that tridents only deal as much damage as an unenchanted bow and this cannot be improved at all other than via Impaling, which is exclusive to aquatic mobs; notably, Drowned do not count as aquatic mobs. Only the harmless squids, salmon, cod, tropical fish, and the less harmless pufferfish and guardians count as "aquatic". Power is considered incompatible with Sharpness and Impaling. Fabrication
Enchantment Tweak Tridents Accept Sharpness: Allows tridents to accept the Sharpness enchantment, increasing their melee damage. See above for justification. Tridents deal 1 more damage than a Netherite Sword, but this tweak only allows them to accept Sharpness; no Smite, no Looting, etc. Sharpness is considered incompatible with Power and Impaling. Fabrication
Potion Tweak Fire Aspect II: Potions of Fire Resistance only protects against fire. They have to be brewed with glowstone dust to also protect against lava. Fabrication
Worldgen Tweak No Dinnerlava: Disables the generation of 1 block hidden lava pockets in the Nether. They were initially added to discourage stripmining in the Nether for quartz, but now that Netherite exists, which can only be reasonably obtained via stripmining, it's just frustrating for no real reason. Fabrication
Structure Tweak huskSpawningInTemples: Only husks spawn in desert temples. Carpet
Structure Tweak straySpawningInIgloos: Only strays spawn in igloos. Carpet Extra
Entity Tweak No Heavy Minecarts: Chest and hopper minecarts will not be slowed down proportionally to the number of items in them. Fabrication
Entity Tweak Furnace Minecart Accepts Any Fuel: Allows furnace minecarts to accept any furnace fuel, rather than just coal and charcoal. Fabrication
Entity Tweak Toggleable Furnace Carts: Powered rails stop furnace carts when appropriate and conserve their fuel. Fabrication
Entity Tweak Enhanced Moistness: Entities are considered "wet" for 5 seconds after leaving a source of wetness. Additionally, lingering or splash water bottles inflict wetness. Also makes wet entities drip to show they're wet. Affects various vanilla mechanics including fire and undead burning. Touching lava instantly removes wetness, meaning lava blades still work. Fabrication
Mob Tweak Spiders Can't Climb Glazed Terracotta: Makes spiders unable to climb glazed terracotta, to make farming them less of a chore if you go to the trouble to make glazed terracotta. Logic: Slime blocks can't stick to glazed terracotta. Are spiders really stickier than solid slime? Fabrication
Mob Tweak Wool Protected Sheep: Sheep wool slightly reduces some types of damage. Fabrication
Mob Tweak Ender Dragon Always Spawns Dragon Egg: Ender dragon always spawn a dragon egg when killed. Fabrication
Mob Tweak persistentParrots: Parrots don't get of your shoulders until you receive proper damage or go in water. Carpet
Mob Tweak renewableSponges: Guardians turn into Elder Guardian when struck by lightning. Carpet
Mob Tweak chickenShearing: Chickens can be sheared to get feathers. Beware! every time u shear a chicken, it gets damaged! Baby chickens can't be sheared. Also works with dispensers. Carpet Extra
Mob Tweak renewableWitherSkeletons: Skeletons turn into wither skeletons when struck by lightning. Carpet Extra
Mob Tweak spiderJockeysDropGapples: Spider jockeys have a 30% chance to drop enchanted golden apples. Carpet Extra

Visual/Audio

Type Description Mod
UI Tweak Use Player List Name In Tag: Changes player name tags to match names in the player list. Good in combination with nickname mods. Fabrication
Visual Tweak No Night Vision Flashing: Disables the flashing effect when Night Vision is about to run out. Makes it just slowly fade out instead. Fabrication
Visual Tweak Omniscent Player: The player render in the inventory follows your cursor, even if it's not inside the game window. Fabrication
Visual Tweak Fullres Banner Shields: Makes shields use the full-res banner patterns instead of weird smaller versions. Compatible with custom patterns and resource packs. Fabrication
Visual Tweak Legible Signs: Adjusts sign text colors to be less garbage. Fabrication
Visual Tweak Less Annoying Fire: Makes the "on fire" overlay half as tall, and removes it completely if you have Fire Resistance. Fabrication
Visual Tweak Normal Fog With Night Vision: Disables the fog brightening effect with Night Vision. Fabrication
Visual Tweak More Explosion Particles: Adds smoke and poof particles to explosions. Fabrication
Visual Addition Note Block Notes: Tells you the note the note block has been tuned to when tuning it or playing it manually above your hotbar. Also shows the octave, and shows the correct note for the snare and click instruments. Fabrication
Audio Tweak Alt Absorption Sound: Taking damage that is completely absorbed by Absorption plays a different (custom) sound. Fabrication
Audio Tweak Long Level Up Sound At 30: Plays the old longer level up sound when you reach level 30. Fabrication
Audio Tweak Note Blocks Play On Landing: Makes note blocks play when landed on. The higher the entity fell, the louder the note will be. Also triggers Observers. Fabrication
Audio Tweak Disable Equip Sound: Disables the unnecessary "Gear equips" sound that plays when your hands change, and is often glitchily played every tick. Armor equip sounds and other custom equip sounds remain unchanged. You won't even notice it's gone. On client, mutes it just for you. On server, prevents the sound from playing at all for everyone. Fabrication
Audio Tweak Oof: Brings back the old "Oof" hurt sound. Unlike the resource pack approach, this is player-specific rather than replacing the generic fleshy damage sound, so it won't result in random things Oof-ing. Fabrication
Audio Addition Ender Pearl Sound: Ender pearls play enderman teleport sound when breaking. Fabrication
Audio Addition Ghast Panic: Makes ghasts randomly play the unused "scream" sound when outside of the Nether. Fabrication
Visual/Audio Addition Cracking Spawn Eggs: Spawn eggs spawn cracking particles and play a sound when used. Fabrication

Technical

Type Description Mod
Performance Uncap Menu FPS: Removes the hardcoded 60 FPS cap in menu screens, instead using whatever you have the framerate cap set to. Fabrication
Performance lagFreeSpawning: Spawning requires much less CPU and Memory. Carpet
Performance optimizedTNT: TNT causes less lag when exploding in the same spot and in liquids. Carpet
Collision Collision Based Landing Pos: More accurately determines what the player is standing on. e.g. with this you can't take fall damage on a slime block while landing on an edge with air next to it. Fabrication
Commands /i And More: Adds /i, /item, /more, and /fenchant commands. /i and /item are shorthand for /give to yourself, and /more increases the size of your held item's stack. /fenchant is like /enchant but it ignores all restrictions. Fabrication
Commands Legacy Command Syntax: Allows numeric arguments to /gamemode and /difficulty, re-adds /toggledownfall, allows TitleCase arguments to /summon, allows numeric arguments to commands that accept items like /give, and re-adds shorthand to /xp. Fabrication
Commands Lenient Command Suggestions: Command suggestions for modded ids will still appear even if the id namespace is not explicitly specified. Fabrication
Commands /mods Command: Adds a /mods command listing all mods on the server. Hovering over the mods shows their descriptions, clicking on them takes you to their homepage if they have one. If Bukkit is not present, also adds a /plugins command suggesting people use /mods instead. Fabrication
Commands Taggable Players: Allows you to assign or unassign various "tags" from players via /fabrication tag. Click the link to learn more. Fabrication
Commands commandDistance: Enables /distance command to measure in game distance between points. Carpet
Commands commandDraw: Enables /draw commands for OPs, allowing to build shapes that are normally difficult to build (circles, spheres, etc.). Carpet
Commands commandInfoEnables /info command for blocks. Carpet
Commands commandPerimeterInfo: Enables /perimeterinfo command that scans the area around the block for potential spawnable spots of the specified mob. Carpet
Commands commandPlayer: Enables /player command for OPs, allowing to control/spawn "players". Carpet
Commands commandProfile: Enables /profile command to monitor game performance. Carpet
Commands commandSpawn: Enables /spawn command for OPs, enabling spawn tracking of mobs. Carpet
Commands commandTick: Enables /tick command for OPs to control game clocks. Carpet
Commands commandPing: Enables /ping for players to get their ping. Carpet Extra
Chat Functionality KillMessage: Adds a KillMessage NBT tag to entities and items, allowing them to show a custom death message when they kill a player. Click the link to learn more. Fabrication
Chat Functionality Linkify Urls: Links in chat are clickable. Fabrication
Chat Functionality Chat Markdown: Adds bold, strikethrough, underscore and italic markdown to chat. Fabrication
Chat Functionality RMB Clears Text Fields: Right Mouse Button will clear text fields. Fabrication
Controls Toggle Sprint: Adds a "Toggle/Hold Sprint" keybind. Fabrication
Controls Toggle Stance: Adds a "Toggle Stance" keybind to switch between standing, sneaking and crawling. Fabrication
Debug Info Mob IDs: Makes entities show their entity ID as a nametag in Creative when F3 is up. Fabrication
Server Ping Privacy: Sends no ping data to IPs that have not successfully logged in in the last 7 days. Prevents scraping. Fabrication
